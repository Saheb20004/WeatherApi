# Weather App

A simple, responsive weather application that displays current weather information for any city using the WeatherAPI service.

## Features

- Real-time weather data fetching
- Clean and responsive user interface
- Temperature display in Celsius
- Weather condition descriptions
- Error handling for invalid city names
- Air quality index data integration

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- WeatherAPI.com API

## Setup

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Enter a city name and click "Get Weather" to see current weather information

## API Configuration

The app uses WeatherAPI.com with the API key already configured. For production use, consider:
- Moving the API key to environment variables
- Implementing proper API key security measures

## Usage

1. Enter the name of any city in the input field
2. Click the "Get Weather" button
3. View the current temperature and weather conditions
4. Error messages will display for invalid city names or network issues
